Schedule for XFC 2025

These are the important dates for XFC 2025.

Date

Event

Description

10/18/2024

Kick-Off

Kick-off meeting introducing the organizers, competition challenge, timeline, etc.

10/25/2024

Seminar: Kessler Introduction

An introduction on how to create and run scenarios in the open-source Python game "Kessler".

11/8/2024

Seminar: Implementing Fuzzy with Kessler

Learn how to implement a Fuzzy System to control ship actions in Kessler.

11/22/2024

Seminar: Implementing Gradient-free optimization in Kessler

Learn how to implement gradient-free optimization algorithms for optimizing controller/FIS parameters in Kessler.

TBD

Help Sessions (as needed)

Help sessions for Python coding, Kessler, respository management, etc. will be held as needed/requested by participants.

1/13/2025

Spring Kickoff and Help session

Primarily an opportunity for participants to check-in with organizers for a Q&A and other topics of discussion related to agent development, submission timelines, etc.

Mid March 2025 (TBD)

Qualifying Deadline

Teams must submit a non-final version of their agents to organizers in order to ensure proper integration ahead of the competition. No performance metrics are used aside from ones about controller evaluation times and ensuring no errors are present.

Mid April 2025 (TBD)

Final Agent Submission Deadline

Teams must have a branch in their project repository for their final agent and share it with the organizers. Integration will happen in this time before the competition to ensure no errors are present.

Late April 2025

Competition

Final Competition day! Teams will make presentations (template/requirements will be given) to show what they did. All submitted agents will then be evaluated live in a playoff format. The playoff will be livestreamed on a streaming service so others can view as well.
