Schedule for XFC 2023

These are the important dates for XFC 2023.

Date

Event

Description

10/02/2023

Kick-Off

Kick-off meeting introducing the organizers, competition challenge, timeline, etc.

10/06/2023

Seminar: Fuzzy Inference Systems

Come learn what this Fuzzy Logic stuff is all about. We'll talk about Fuzzy systems, trees, terminology, python modules, and everything in between.

10/16/2023

Seminar: Genetic Algorithms and other Gradient free optimization

Seminar on gradient-free optimization methods (GA, PSO, etc.) and their application to fuzzy systems.

10/23/2023

Seminar: Kessler Game

Introduction to the new Kessler Game and Optional Q&A session

10/XX/2023

Seminar: Explainability in AI / Fuzzy Systems

Dr. Scott Dick from the University of Alberta will give his perspective on explainability in AI/Fuzzy systems.

10/XX/2023

Seminar: Gradient-based Reinforcement Learning

Dr. Barnabas Bede, from Digipen Institute of Technology will give a seminar on gradient-based reinforcement learning methods for fuzzy Systems.

11/13/2023

Competition Intro

Introduction to the competition; rules, scoring, format and beginning work on solutions.

01/08/2024

Spring Semester Kickoff

Spring Kickoff and Optional Q&A session

03/01/2023

Milestone Submission

We are including some milestones this year, some optional some required, that will be submitted by this date.

03/08/2024

Qualifying

Teams must qualify for final competition by having an agent that is able to function within the game environment. Performance is not considered, only integration and code functionality-related aspects.

03/25/2024

Final Code Submission

Teams must submit all code for final trained agents by 10pm EST.

03/29/2024

Competition

It's game day! Students will present their work followed by the explainability evaluation and then the competition scenarios!
