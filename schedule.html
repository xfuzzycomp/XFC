Schedule for XFC 2023

These are the important dates for XFC 2023!

Schedule for 2022/2023 Competition
Date Event Description
8/29/2022 Kick-Off Kick-off meeting introducing the organizers, competition challenge, timeline, etc.
9/6/2022 Game Intro We'll introduce you to the game, Asteroid Smashers, repos, structure, etc.
9/12/2022 Seminar: Fuzzy Inference System (FIS) Come learn what this Fuzzy Logic stuff is all about. We'll talk about Fuzzy systems, trees, terminology, python modules, and everything in between.
9/26/2022 Seminar: Explainability in Fuzzy Systems Dr. Scott Dick from the University of Alberta will give his perspective on explainability in AI/Fuzzy systems.
10/3/2022 Seminar: Genetic Algorithms Seminar on gradient-free optimization methods (GA, PSO, etc.) and their application to fuzzy systems.
10/XX/2022 Seminar: Gradient Descent for Fuzzy Systems Seminar on gradient descent for fuzzy Systems by Dr. Barnabas Bede.
10/XX/2022 Seminar: Gradient-based Reinforcement Learning Seminar on gradient-based reinforcement learning methods for fuzzy Systems by Dr. Barnabas Bede.
11/07/2022 Competition Intro Introduction to the competition; rules, scoring, format and beginning work on solutions
01/17/2023 Spring Semester Kickoff Welcome back for the spring semester! Optional help / Q&A session
03/06/2023 Qualifying Teams must qualify for final competition by having an agent that is able to function within the game environment. Performance is not considered, only integration and code functionality-related aspects.
03/27/2023 Final Code Submission Teams must submit all code for final trained agents by 10pm EST.
03/31/2023 Competition It's game day! Students will present their work followed by the explainability evaluation and then the competition scenarios!
