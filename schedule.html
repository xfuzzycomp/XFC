Schedule for XFC 2023

These are the important dates for XFC 2023!

Schedule for 2022/2023 Competition
Date Event Description
9/7/2022 Kick-Off Kick-off meeting introducing the organizers, competition challenge, timeline, etc.
9/13/2022 Game Intro We'll introduce you to the game, Asteroid Smashers, repos, structure, etc.
9/20/2022 Fuzzy Seminar #1 Come learn what this Fuzzy Logic stuff is all about. We'll talke about Fuzzy systems, trees, terminology, python modules, and everything in between.
9/27/2022 Fuzzy Seminar #2 Jerry Mendel's perspecive on the explainability of fuzzy systems.
10/4/2022 Optimization Seminar Seminar on gradient-free optmization methods (GA, PSO, etc.) and their application to fuzzy systems.
10/18/2022 Help Session Optional help session for game (Asteroid Smasher), OOP, etc.
11/08/2022 Competition Intro Introduction to the competition; rules, scoring, format and beginning work on solutions
01/18/2023 Spring Semester Kickoff Welcome back for the spring semester! Optional help / Q&A session
03/07/2023 Qualifying Teams must qualify for final competition by having an effective solution f or a simple Asteroid Smasher scenario (score minimum, don't die within 10 s, etc.). Code runs without crashing
04/04/2023 Final Code Submittal Teams must submit all code for final trained agents by 10pm EST.
04/09/2023 Competition Its game day!
